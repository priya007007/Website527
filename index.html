Reinforcement Learning Based Autopilot

Establishing autonomous flights will have big consequences for the future and the way the aviation industry works. Keeping this in mind, we will explore the possibility of autonomous air transport using Machine Learning Algorithms.

About

The goal of this project is to be able to train Autopilot RL agents using neural networks and Policy Gradient algorithms. The proposed autopilot should be able to perform various aircraft maneuvers such as descent/ascent and left/right turns as required by a given flight plan.
