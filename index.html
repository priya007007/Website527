Deep Learning Based Autopilot System using X Plane 11

Establishing autonomous flights will have big consequences for future and the way the aviation industry works. Keeping this in mind, we will explore the possibility of autonomous air transport using Machine Learning Algorithms
Goal:To be able to train Autopilot RL agents using neural networksand Policy Gradient algorithms.

What is expected

The goal of this project is to be able to train Autopilot RL agents using neural networksand Policy Gradient algorithms. The proposed autopilot should be able to performvarious aircraft maneuvers such as descent/ascent and left/right turns as required bya given flight plan.
